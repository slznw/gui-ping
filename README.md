# gui-ping
Python program that uses basic TCP/IP command and features cool UI

## Requirements 
+ python 3
+ Windows 10/11 or macOS 10.14+ (untested and may not work on any linux distro)

## How it works?
My code is pretty simple: it uses tkinter for gui, os for ping and sys for OS definition. 

Note that notifications only works on macOS and won't be ported to windows due to the lack of native notification functionality
